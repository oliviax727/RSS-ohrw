Welcome

This page is for an RSS News that I have made for myself. This webpage contains various news feeds ranging from political news to scientific publication. All for the purpose of staying informed. While RSS is an ancient internet feature. I think it's better to use than most news aggregator sites and services. Sites like MSN, news.com, Ground News, Google News, etc. — they all take control away from the user and often present either sensationalism or false bias labelling. This central location also can help me aggregate news on scientific publications.

How to use the Newsreader

Navigation

The newsreader has three (two) buttons: the menu button opens the sidebar menu, the reload button will reload the feed, and the expand/minimise button will expand/minimise the window. On each feed item, there is a 'Read More ...' button and 'Dismiss Item' button; clicking the former will open a new tab, redirecting you to the article website and will mark the post as 'read'; clicking the latter will dismiss the item, moving it to the bottom of the feed.

Feed

Each feed in the navigator ribbon will contain information about the current date and, when in desktop mode, information about the current section will be displayed as well. The newsreader in each section will only show stories from the websites/feeds that correspond to that sections name (see the sidebar for more information). Section names are rather self-explanatory. Each section is additionally colour-coded for both stylistic purposes, and as a visual aid to differentiate sections (however it is not required to rely on, for the sake of accessibility). Note on scrolling: when in full-screen on mobile, there may be difficulty with seeing entries at the bottom of the feed. This is due to the mobile browser you use hiding information behind the lower navigation bar. You will need to purposefully hide it either through settings or fiddle with scrolling to get it to disappear. The primary feed is an aggregate of the latest stories (5 per section). Dismissing any item anywhere will stop it from showing in the main feed (unless it is undismissed from another feed in a different window).

Items

Each item always contains: a title, a description, a channel title, a channel name, and a link to the page (by clicking the 'Read More ...' button). When the page is in desktop mode, a channel image will also be displayed. Feed items may additionally include publishing dates.
